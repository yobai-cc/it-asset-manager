# IT 固定资产管理系统

## 项目简介

面向 IT 管理员的固定资产追踪系统。覆盖资产全生命周期管理：入库、分配、调拨、维修、报废，同时提供员工花名册、二维码标签打印、移动扫码查询、打印机墨粉管理等功能。

## 业务场景

- IT 管理员维护资产台账，管理分配/归还/调拨/报废流程
- 员工花名册作为资产分配对象池，普通员工默认不需要登录系统
- 物理标签打印（60×40mm QR 标签）贴在设备上，扫码即可查看资产信息
- 打印机墨粉库存跟踪与更换成本分析

## 技术栈

| 层次 | 技术 |
|------|------|
| 后端 | Python 3.11 / Flask 3.1 |
| 数据库 | SQLite（原生 SQL，无 ORM） |
| 前端 | Jinja2 模板 + 原生 JavaScript + CSS |
| 认证 | Flask session + Werkzeug 密码哈希 |
| 打印 | 浏览器原生打印，立象 Argox 热转印打印机 |
| 测试 | pytest，202 个自动化测试 |

无前端框架、无构建步骤、无 Docker 依赖。

## 项目规模

| 文件 | 行数 | 职责 |
|------|------|------|
| server.py | ~2600 | 所有路由 + API（单文件） |
| models.py | ~480 | 数据模型、常量、Schema、迁移 |
| init_db.py | ~170 | 数据库初始化 + 种子数据 |
| style.css | ~2600 | Design Tokens + 全局样式 |
| test_api.py | ~3000 | 202 个自动化测试 |
| templates/ | 20 个模板 | 12 管理页 + 4 员工页 + 4 公共页 |

## 部署

```
Flask (0.0.0.0:5000) → Caddy 反向代理 (10.18.0.68:9090)
```

```bash
# 开发
pip install -r requirements.txt
python3 init_db.py       # 初始化数据库
python3 server.py        # 启动服务

# 生产建议 systemd + gunicorn
```
